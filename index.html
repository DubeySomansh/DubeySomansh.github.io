Projects

CF-Stress — Concurrent Online Judge

CF-Stress is a concurrent online judge built in Go with Redis, designed to stress-test competitive programming solutions under large randomized and adversarial input distributions.

The system supported hundreds of concurrent users during live contest environments, executing sandboxed submissions in parallel while enforcing strict resource and runtime constraints.

Autonomous Ground Vehicle — Team VISiON

As part of Team VISiON at IIT Kanpur, I worked on perception and planning pipelines for an autonomous ground vehicle operating in dynamic environments.

The project involved CNN-based object detection using ROS and OpenCV, along with classical path-planning algorithms such as Dijkstra and A*, integrated into a closed-loop navigation system.
